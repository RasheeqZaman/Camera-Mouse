# Camera-Mouse

## About

Camera Mouse detects hand position from the webcam and moves the mouse cursor using the YOLOv3 algorithm in Python.

## Features

The software lets you:
- Use mouse cursor without a mouse or trackpad.
- Completely ad-free.
